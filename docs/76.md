# 2025年11月8日-2025年11月15日


## 1.NCE-Flow

这是一个开源的新概念英语在线点读项目。

它的功能很简单，打开之后选择一个要听的单元，

点击播放，它就会连续播放整个单元的内容，

也可以单独点击并播放某一句话。

如果你在学新概念英语，有可能会有帮助。


NCE-Flow（新概念英语）
https://github.com/luzhenhua/NCE-Flow

## 2.copyparty


这是一个开源的文件服务器，

它支持上传、下载、分享文件，支持断点续传、边传边下，

你可以把它理解成是一个小网盘，并且能真的当成网盘使用。

并且，这个项目使用起来非常简单，

因为整个项目只有一个核心文件，直接把它跑起来即可。

它的兼容性也很变态，

从 Python 2.7，到最新的 Python3，各种电脑系统、手机系统都能部署。

从上个世纪的 IE 4，到各种游戏机，只要有浏览器就能访问。


除此之外，它还有很多很多功能，

正因为简单又好用，所以仅凭一个文件，它就获得了 30 多 k Star。


copyparty（文件服务器）
https://github.com/9001/copyparty

## 3.open-source-games


这个项目中，收藏了一大堆开源的游戏，

覆盖了目前最火的十几个类型，

从角色扮演、到 FPS、塔防游戏都有，

甚至里边有古墓丽影、GTA 等游戏的开源版本。

因为是开源的，所以它们也都是免费的，

网游类型的游戏，还可以自己部署。



open-source-games（开源游戏）
https://github.com/bobeff/open-source-games

## 4.NinjiaTag

前几年，苹果推出了一个叫 AirTag 的定位设备，

可以把它挂在各种物品上，防止东西丢掉。

NinjiaTag 是一个类似的开源项目，

可以让你自己 DIY 一个兼容 FindMy 的定位设备，

并且，相比于 AirTag，它还可以长期记录运动轨迹。


NinjiaTag（定位）
https://github.com/zhzhzhy/NinjiaTag-backend

## 5.helium


这是一个开源的电脑浏览器，兼容三大操作系统，

它默认就开启了 广告拦截、屏蔽追踪器、禁止第三方 Cookie 等选项。

同时，它删除了其它浏览器中臃肿的部分，更加轻量、占用资源也会低一些。

除此之外，我还很喜欢它的分屏功能，可以同时并排打开两个界面。


helium（浏览器）
https://github.com/imputnet/helium

## 6.Xiaomi Miloco

小米开源了一份智能家居未来探索方案，

它以米家摄像机为视觉信息来源，

以大模型为核心，可以通过自然语言交互，完成规则设置、设备的复杂指令控制，

能自动完成家庭生活中的各类智能需求，支持各种米家设备，

感兴趣的可以了解一下。


Xiaomi Miloco（米家）
https://github.com/XiaoMi/xiaomi-miloco

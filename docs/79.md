# 2025年11月29日-2025年12月6日

## 1.WSABuilds

WSA，是 Windows Subsystem for Android 的简称，

可以让我们在 Windows 上，像原生应用一样，无缝的使用安卓应用。

这本来是微软官方推出的一个功能，

但是在今年三月，因为收入不佳已经停止维护。


WSABuilds 是一个基于 WSA 的开源项目，

并且，它在 WSA 的基础上，进行了优化。

比如，支持 Play 商店，和 Root 权限。


虽然微软已经不提供更新了，但这仍然是在电脑上，使用手机软件的最佳方式之一，

如果你有此类需求，可以尝试一下。


WSABuilds
https://github.com/MustardChef/WSABuilds

## 2.noMeiryoUI


这是一个可以修改 Windows 系统字体的开源项目，

在多年前，Windows 曾经自带过这个功能，

但是，Win 8.1 之后就被删除了。


noMeiryoUI 支持一键替换全部场景的字体，

也支持只设置部分项目。


现在屏幕上，展示的是使用默认字体的效果，

这个是我修改为苹果字体的效果，

区别还是很明显的。


noMeiryoUI
https://github.com/Tatsu-syo/noMeiryoUI

## 4.chrome_installer

如果你在 Chrome 的官网下载 Chrome ，

获得的会是一个空壳的安装器，使用的时候，

还需要等它自动下载完整版本，非常不方便。

如果网络不好，或者是在内网环境使用，可能很久都跑不起来。


chrome_installer 这个项目，提供了完整的 Chrome 安装包，

双击就能直接打开使用，

如果你需要，可以收藏一下。

chrome_installer
https://github.com/Bush2021/chrome_installer


# 2025年11月1日-2025年11月8日


## 1.RyTuneX

这是一个开源的 Win10、Win11 优化工具，

打开之后，首页会显示目前电脑资源的使用状态，

我们选择 “优化” 页面，可以在这里看到很多选项，

比如 “禁用自动更新”、“禁用菜单显示延迟”、“启用游戏模式”。

“修理” 页面中，可以获取到电池健康报告、对内存进行诊断，优化电脑硬盘。

“精简” 页面可以一键批量卸载电脑上的各种软件，比电脑自带的卸载页面要好用很多。

“隐私” 页面可以关掉各种遥测和数据收集。

“功能” 中可以调整很多系统功能。

总而言之，这是一个非常全面的一站式优化方案。

RyTuneX（电脑优化）：https://github.com/rayenghanmi/RyTuneX

## 2.azerothcore-wotlk


这是一个由玩家制作的开源魔兽世界服务器，

它复刻了魔兽非常经典的 “巫妖王之怒” 版本，

并且部署起来非常简单，只需要一条 Docker 命令就能跑起来，

然后你就有了一个属于自己的魔兽世界服务器，

搭配本地的客户端，就可以玩起来了。

azerothcore-wotlk（开源魔兽服务器）：https://github.com/azerothcore/azerothcore-wotlk

## 3.qq-chat-exporter

我之前分享过很多微信数据导出工具，很多人都问 QQ 有没有类似的软件。

qq-chat-exporter 就是一个这样的开源项目。


它可以一键导出 QQ 的聊天记录，

包括 消息、图片、视频、文件、表情包 等各种格式的内容，

并且使用起来非常简单。


拿 Windows 版本来说，双击 launcher-user.bat 文件运行起来，

然后扫码登录，在浏览器中打开它的 Web 页面，输入终端中给出的访问令牌。


点击 “新建任务”，选择一个群或者好友，

设置一下要导出的聊天记录格式和时间等选项，

最后点击创建任务，聊天记录数据就被导出了。


不同格式的数据有不同的用处，可以拿来永久保存，或者当做分析、训练数据。


qq-chat-exporter（QQ记录导出）：https://github.com/shuakami/qq-chat-exporter

## 4.MuJing

这是一个开源的英语学习项目，

它的特点是，除了常见的背单词、拼写单词之外，还增加了电影、电视剧的场景，

每一个单词都有对应的影视片段，

通过原汁原味的场景，给你带来更深刻的记忆。


MuJing（影视片段学英语）：https://github.com/tangshimin/MuJing

## 5.snake

这是一个开源的贪吃蛇，

和普通的贪吃蛇不一样的是，它是通过浏览器的 URL 来玩的，

snake（地址栏玩贪吃蛇）：https://github.com/epidemian/snake

## 6.MagicMirror

这是一个开源的魔镜项目，

简单来说，

只需要一块儿单向透视玻璃镜子，加上一个树莓派，和一定的动手能力，

你就能自己做出来一个这样的智能镜子。


假如有代码能力，甚至可以给这块镜子定制很多功能，

没有代码能力，也可以用社区中已经开发好的模块儿，实现各种神奇的效果。


目前，这个项目已经有了一个非常庞大的社区，

如果你感兴趣且正好有时间，就可以动手给自己做一个，并加入到社区中添砖加瓦。


MagicMirror（魔镜）：https://github.com/MagicMirrorOrg/MagicMirror

# 2026年3月22日-2026年3月28日

# 1.Skills

本周在 GitHub 上火了好几个奇怪的skills，比如这个叫 PUA 的 skills，它整合了十几家大厂的话术和绩效考核体系。

当AI执行任务失败时，他会问 AI 你连这个都解决不了，我怎么给你打绩效？让AI切换不同方式尝试。

再次失败时会问 AI 你的底层逻辑是什么？顶层设计在哪儿？抓手在哪儿？让AI去网上找资料和读源码。

如果一直失败，他会告诉AI别的模型都能解决，你可能就要毕业了，开启AI的拼命模式。

这个叫 PUAClaw 的项目更加离谱，一共总结了16套对AI施压的话术，从最温和的对AI拍马屁，让AI角色扮演到情感绑架、道德绑架，甚至直接对AI进行各种威胁。并且这里面的所有内容都不是随便写的，而是每一种都有作用。

机制研究以及参考文献，除了 PUA 流派，还有关爱派。

比如这个叫 NOPUA 的项目，他认为在同样的方法论下，对AI施压容易起到反向效果，关心鼓励和爱反而更有用。

虽然这些项目看起来像是在恶搞，但实际上各种话术只是表象，在底层是一套让 AI 不轻易放弃，有能力不放弃，主动出击的方法论，都是经过工程验证确实有效的。

PUAClaw：https://github.com/puaclaw/PUAClaw
pua：https://github.com/tanweai/pua
nopua：https://github.com/wuji-labs/nopua

# 2.project-nomad

有人把目前人类的各种技术知识做成了一个末日工具包。

这里包含离线的维基百科、医学知识、求生指南、电子书，一个小型的AI模型助手，离线的地图数据分析工具。

如果你认为世界末日快来了，那可以收藏一下。

project-nomad：https://github.com/Crosstalk-Solutions/project-nomad

# 3.keyStats

这是一个可以统计你键盘敲击次数、鼠标点击次数、鼠标移动滚动距离的开源项目。

支持 Windows 和 Mac 系统默认情况下会显示当天的统计数据，也可以查看过去七天、30天所有时间的数据，还能按照应用来统计。

除了 keyStates，还有一个叫 Work_Review 的项目，它的功能是记录每天使用电脑的时长，访问网站的时长，使用应用的时长，以及通过自动截图OCR来记录使用电脑时的时间线。

最后再通过AI来生成日报和总结工作内容。这种软件我个人的使用需求不大，但如果你想要统计自己的工作、游戏情况，或许会有作用。

keyStats：https://github.com/debugtheworldbot/keyStats
Work_Review：https://github.com/wm94i/Work_Revieweditor

# 4.cheatreader

这是一个电脑端的本地阅读器。

作为一个普通阅读器，它平平无奇，但是打开透明模式，调整完大小之后，我们就得到了一个摸鱼神器。

cheatreader：https://github.com/yaoyao2mm/cheatreader

# 5.weclaw

微信开放了 ClawBot，可以对接各种龙虾工具。

但是既然开放了接口，那么用户到底用它接入什么，那就不是微信官方说了算了。

所以一周内就出了很多衍生项目，比如这个 weclaw 和这个 wechatbot 可以让我们接入各种 AI agent。

未来我相信依据此接口的玩法会越来越多，但是这也是微信的目的，用来聊什么无所谓，是不是在微信聊才重要。

weclaw：https://github.com/fastclaw-ai/weclaw
wechatbot：https://github.com/corespeed-io/wechatbot

# 6.editor

这是一个开源的房屋建模工具，可以让我们在网页上创建3D的房屋建筑物模型。

根据我的体验来看，它的基础功能很齐全，并且使用起来非常流畅。

editor：https://github.com/pascalorg/editor

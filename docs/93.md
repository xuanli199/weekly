# 1.wsl2-distro-manager

如果你是一个计算机爱好者，那么可能会经常需要用到 linux。

如果不想做服务器，那么 windows 自带的 linux 子系统 WSL 就是最好的选择之一。

不过WSL要使用命令行来操作，虽然对于 linux 用户来说并不是什么难题，但也不是很方便。

WSL2-distro-manager 是一个开源的 WSL 管理器，可以通过简单直观的页面来使用 WSL 的各种功能，一键在 windows 上创建 linux 子系统，一键关闭、一键启动，一键保存成模板，非常方便。

wsl2-distro-manager：https://github.com/bostrot/wsl2-distro-manager

# 2.CCTV-Smartphone-AI-Monitoring（Sentinel）

这是一个可以把你闲置的安卓手机变成监控摄像头的开源项目，并且它可以搭配视觉模型进行实时的识别和预警。

那么它能做什么呢？

假如家里老人行动不便，可以拿来监控老人起身摔倒或者有其他风险的场景。

对准办公室门口，可以监控老板是不是来了。

家里有宠物，可以监控他是否去了厨房。

CCTV-Smartphone-AI-Monitoring（Sentinel）：https://github.com/suzuran0y/CCTV-Smartphone-AI-Monitoring

# 3.pi-mono

这是一个用来构建 AI agent  的开源项目，它其中包含了多个工具包，比如这个coding agent 就是最近最火的 agent 产品 openclaw 底层引擎。如果你也想开发 agent 产品，那这个龙虾同款引擎或许会对你有用。

pi-mono：https://github.com/badlogic/pi-mono

# 4.OpenClaw-Docker-CN-IM

看名字就知道这个项目是干什么的。它是一个打包了 OpenClaw 和国内各大聊天软件的 Docker 镜像。

如果你不想用大厂的各种一键包，那么使用它在部署的时候也能给你省非常多的时间。

并且因为是使 Docker 刀部署，所以安全性也能提高很多。

如果想在本地电脑上使用，那么可以试一下这个叫 ClawX 的开源项目。

OpenClaw-Docker-CN-IM：https://github.com/justlovemaki/OpenClaw-Docker-CN-IM
ClawX：https://github.com/ValueCell-ai/ClawX

# 2025年12月6日-2025年12月13日

## 1.Open-AutoGLM


在豆包手机爆火之后，智谱开源了它们的手机端智能助理框架 AutoGLM，

简单来说，它可以像豆包手机一样，

你只需要用嘴下命令，AI 就会自动解析你的需求，打开对应的软件进行操作。

它会自动解析软件界面上的内容， 并自主完成规划。


比如我之前体验的场景：


想要挑一个礼物，

给它下任务之后，它会自动搜索资料、确定挑选策略、让我选择孩子的信息。

最后到购物软件搜索，反复查看各种商品的介绍、顾客的评价，

并且还会换关键词多次搜索，最后给到一个清单。


整体来说，因为不是系统应用，所以没有豆包手机那么流畅，

但同样具有非常高的可玩性。


Open-AutoGLM（AI操作手机）
https://github.com/zai-org/Open-AutoGLM

## 2.cloudflare-error-page


因为最近 Cloudflare 经常发生大面积崩溃，

各种网站动不动就会看到类似这样的报错。


所以，有人就以此为灵感，开发了这个项目，

它的功能非常简单，

可以让你自己根据需要，定制一个仿 Cloudflare 风格的错误界面。


然后，把这个界面放到你自己的网站里，替换掉默认的报错提醒，

以后，如果你的网站崩了，就可以把锅甩给 Cloudflare。


因为看到报错的人，都会认为是 Cloudflare 出问题了。
 

cloudflare-error-page（自定义错误页）
https://github.com/donlon/cloudflare-error-page

## 3.RemoveWindowsAI


看名字就知道，它是做什么的，

AI 爆火之后，微软在 Windows 中，塞入了大量的 AI 功能。

有的人可能会用到它们，但也有很多人完全不需要，却还要被它们占用资源。

RemoveWindowsAI 可以让你通过一行命令，

禁用、或者删除掉 Win11 中的 AI，并且能防止在更新时再次自动安装。


它还提供了一个后悔药 “备份模式”，

如果后续还有可能会使用这些 AI，可以把它打开。


RemoveWindowsAI（删除AI）
https://github.com/zoicware/RemoveWindowsAI

## 4.gpu-hot

这是一个可以实时获取英伟达 GPU 指标的项目，

能压秒级的展示利用率、温度、内存、功耗、风扇转数等等数据。

同时可以通过 API，给第三方应用提供数据。


gpu-hot（显卡数据）
https://github.com/psalias2006/gpu-hot

## 5.LazyCat-Bookmark-Cleaner


如果你经常用浏览器，又恰好是一个收藏癖，

那你的浏览器书签栏中，肯定会有大量的书签。

但其中肯定有很多重复的、被删的、没法正常访问的。


LazyCat 是一个开源的书签整理扩展，

把它安装到浏览器上之后，点击 “扫描书签”，

然后稍微等一会儿，它就会自动扫描收藏的书签，并把它们分类展示，

帮助你快速清理收藏的无用内容。


点击 “书签画像”，还能看到对你收藏的简单数据分析。


LazyCat-Bookmark-Cleaner（书签清理）
https://github.com/Alanrk/LazyCat-Bookmark-Cleaner


## 6.omnibox

如果你经常用浏览器，又恰好是一个收藏癖，

那你的浏览器书签栏中，肯定会有大量的书签。

但其中的绝大部分，可能都只会吃灰，永远都不会看第二次。


omnibox 是一个开源的 AI 知识中枢，

你可以一键把浏览器上，看到的各种网页收藏进去，

也可以在手机中分享、微信中转发内容进去

或者自己手动写、语音录制内容。


然后，这些数据就会变成你的知识库，

因为接入了 AI，所以还能很方便的通过 AI 整理、问答其中的内容。


omnibox
https://github.com/import-ai/omnibox

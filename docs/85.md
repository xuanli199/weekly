## 1.Sandboxie


如果你经常用电脑，可能会遇到过很多不安分的软件，

比如总是想自启、偷偷弹广告、修改你电脑的设置、读取电脑上的文件。

于是，很多人为了安心，会把它们放在虚拟机里边运行，

但其实还有另一种隔离工具，叫做 “沙盒”。


在 Windows 上，自带的就有这个功能，但是它并不好用。


Sandboxie 是一个非常知名的开源沙盒软件，

它使用起来也非常简单。

打开之后，在右上角点击 “沙箱” -> “新建沙箱”，

输入一个名称，沙箱类型一般选择 “标准沙箱” 即可。


之后，我们有两种使用方式，

一种是用沙箱模式，运行电脑上的已有软件，

右键点击沙箱，选择 “运行” -> “运行程序” -> “浏览”，

之后选择已安装软件的启动程序。


第二种是把安装包直接拖入到 Sandboxie 中，

然后选择一个沙箱运行。


需要注意的是，在沙盒中运行软件的时候，所产生的数据也会被放在沙盒里，

所以，如果是网盘类的软件，

需要下载文件之后，右键单击沙盒，在 “沙箱内容” -> “浏览文件” 中手动复制到系统环境。


此外，沙盒默认会清除数据，如果运行 QQ 等需要长时间保存数据的软件时，

需要右键点击沙盒，选择 “沙箱选项”，勾上 “保护此沙箱免受删除或清空”。


除了隔离软件和系统之间的环境，

沙盒也能用于多开软件，只要多创建几个 “沙箱”，然后把软件导入即可。


Sandboxie
https://github.com/sandboxie-plus/Sandboxie

## 2.snow-shot


这是一个开源的截图工具，

一句话总结它的优点，就是 开源、免费、功能非常齐全、且很好用。


长截图、钉图、扫码、添加各种图形、添加水印、添加文字各种功能都有。


还可以通过插件系统，额外添加文本识别、视频录制、翻译、AI 对话等高级功能。


如果你在用 Snipaste、PixPin 或系统自带工具，但是对它们不太满意，

snow-shot 就很值得了解一下。


snow-shot
https://github.com/mg-chao/snow-shot

## 3.myopic_defocus


根据一篇视觉相关专业期刊上的论文显示，

在电脑屏幕上，模拟离焦图像，有减缓近视，增强视力的效果。

荣耀就基于此原理，开发了 “离焦视力舒缓” 功能。

myopic_defocus 是一个开源的浏览器扩展，同样是使用同样的理论打造，

安装之后，可以在浏览器页面上，模拟离焦的视觉效果。

除了这个插件，目前市面上也有此类的软件，感兴趣的可以试一下。


myopic_defocus
https://github.com/refractify/myopic_defocus

## 4.Amarok-Hider


这是一个开源的 安卓软件、文件 隐藏工具，

它的功能很简单，选一下你手机上不想让人看到的软件或者文件，

点击 “隐藏”，这些内容就被藏起来了。

需要的时候，再点击 “取消隐藏”，它们就出现了。


它自带了很多种隐藏模式，比如文件可以通过混淆文件名和签名的方式隐藏，

也通过 Root 直接更改文件权限。


除了隐藏别人， Amarok 本身也能自我隐藏，

可以把它伪装成一个而日历软件，或者直接把图标删掉，只能通过拨号盘打开。

Amarok-Hider
https://github.com/deltazefiro/Amarok-Hider

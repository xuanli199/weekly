# 2026年3月1日-2026年3月7日

# 1.airi

neuro sama 是一个由AI驱动的虚拟主播，在全世界都拥有非常高的热度。

他可以自己玩游戏，可以聊天，可以和直播间互动。

不过因为是个虚拟主播，所以他面向的是整个直播间的观众，并且直播完下线后就会消失。

airi 是一个使用同样理念打造的开源项目，可以让我们轻松运行起来一个只属于自己的虚拟生命。

他同样能聊天，能对话，能陪着玩游戏，还能根据自己的喜好设置他的人设，设置他的形象。

然后我们可以用它来做直播，也可以当成一个二次元的朋友用来陪伴自己。

airi：https://github.com/moeru-ai/airi

# 2.pausemygameW

这是一个开源的游戏暂停工具，听这个描述你可能会有些迷惑，游戏不是自己就能暂停吗？为什么还需要第三方工具？

这是因为 pausemygameW 不仅可以暂停游戏中的内容，还可以释放出游戏占用的 CPU、GPU 资源。

假如你在玩游戏忽然不想玩了，但是为了其他进程能够流畅运行，就需要把它给退掉。

而现在可以把它给暂停，然后去玩别的游戏或者做别的事，等想玩的时候点开它，游戏就会原样恢复。

pausemygameW：https://github.com/cornradio/pausemygameW

# 3.vphone-cli

有人成功解开了苹果的限制，在mac系统上抛弃来了完整的IOS系统，可以安装各种软件，可以批量开启关闭，可以批量脚本控制，并且支持多种系统版本，能越狱。

如果你有批量使用测试IOS的需求，或许会对你有用。

vphone-cli：https://github.com/Lakr233/vphone-cli

# 4.alt-sendme

这是一个用于分享文件的科研项目。

在发送端选择一个文件，然后把获得的票据分享给接收者，接收者粘贴票据就能把文件保存到本地，就像是一个一次性的云盘。

它不像 localsend 之类的软件，必须在局域网内才能传输，而是只要有网就能用，非常方便。

alt-sendme：https://github.com/tonyantony300/alt-sendme

# 5.OpCore-Simplify

黑苹果是指在非苹果官方的硬件上运行苹果的 macos 系统，以前想要自己装一个黑苹果，步骤会非常麻烦。

而现在这个过程已经差不多能自动完成了，OpCore-Simplify 就是一个这样的项目，如果你想试一下 mac 生态，但是不想买苹果的电脑，可以用它搭一个黑苹果系统玩一下。

OpCore-Simplify：https://github.com/lzhoang2801/OpCore-Simplify

# 6.FireRed-OpenStoryline

这是小红书开源的一个视频剪辑工具。

和普通的剪辑软件不一样的是，它是完全由 AI 驱动的，只需要对话就能自动找素材，自动剪辑、自动配乐，自己添加字母花字，看他剪出来的成果。

个人感觉做 vlog 的内容完全没有问题。

FireRed-OpenStoryline：https://github.com/FireRedTeam

# 7.MinecraftConsoles

2014 版我的世界主机版本源码被泄露，目前已经在 GitHub 上开源。

该版本可以被编译出来并在 Windows 上运行，能正常游玩，并且支持多人联机。

MinecraftConsoles：https://github.com/smartcmd/MinecraftConsoles

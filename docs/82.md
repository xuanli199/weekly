## 1.HQ-ICON


作为一个经常分享软件的博主，我经常会在视频中，用到各种 APP 的高清图标，


但是，直接用搜索引擎来搜索，大部分情况下，都只能得到低质量版本。

此前我曾经分享过，

在 App Store 网页版的代码中，获取高清 APP 图标的办法，但是稍微有点麻烦。


HQ-ICON 就是一个可以解决这种问题的开源项目，并且使用起来很简单。

在这里搜一下名字，就能获取到对应的图标，点击一下即可保存到本地。


它的数据同样来自 App Store，所以理论上，App Store 有的软件都能找到。


HQ-ICON
https://github.com/YuKongA/HQ-ICON

## 2.Flyoobe

目前 Win10 已经停止维护，

想要继续获取安全更新，要么得买扩展包，要么就只能升级 Win11。

但是 Win11 并不是想更新就能更新的，它对硬件有很多要求。


Flyoobe 是一个可以让老电脑，也能强制安装上 Win11 的开源项目，

如果你是 Win11 爱好者，那或许会对你有用，

虽然可能并没有 Win11 爱好者这个群体。


Flyoobe
https://github.com/builtbybel/Flyoobe

## 3.pinme


借助于现在的 AI，即使是完全的编程小白，也能够非常轻松的做出来一个网页了。

但是很多 AI，生成的都是这样的静态 HTML 网页。

你可以在本地看，但是想要把它部署到服务器，却不一定能搞定。


就算不是小白，程序员有时候也会遇到做了一个网页，

但一时想不到哪里有免费、且流畅的静态网站服务器。


pinme 是一个开源、无需登录、开箱即用的网页部署工具，

它使用起来非常简单，只需要到官网，把静态的 HTML 上传上去，就能得到一个 URL。

之后就能通过这个 URL 访问了。


不想用官网，也可以通过命令行工具来上传、下线网站。


pinme
https://github.com/glitternetwork/pinme


## 4.staticrypt

使用 pinme 来部署静态网页，虽然很方便，

但是，只要有 URL 的人，都能通过浏览器来轻易的访问它。

实际上，部署在公开服务器上的静态网页，都会有这种 “缺点”。

如果你的网页并不想让那么多人看到，就可以试一下 staticrypt。


这是一个给 HTML 文件一键加密的工具，

用命令行处理之后，再上传到服务器，

在访问时就会要求输入密码。


staticrypt
https://github.com/robinmoisson/staticrypt

## 1.code-box

如果你经常访问 某SDN、某乎、掘金 等网站，

可能总是会遇到不登录账号就 无法查看、无法选中、无法复制 等情况。

code-box 是一个开源的浏览器扩展，可以直接去除这些网站乱七八糟的限制，

想复制就复制、想查看就查看，不会再被烦人的登录弹窗干扰。

code-box（免登录复制）
https://github.com/027xiguapi/code-box

## 2.TrendRadar

这是一个开源的新闻资讯热点助手，

它会持续通过爬虫，自动获取 某音、某乎、某B 等主流平台的热门信息，

然后用 AI 对获取到的信息进行筛选分析，挑出来符合用户要求的内容，

并通过推送系统，把它推送到指定的微信、邮箱、飞书、钉钉等软件。


此外，它还集成了基于 MCP 的 AI 对话分析功能，

能让用户通过自然语言，查询、分析各种新闻内容。


TrendRadar 使用起来也非常简单，不需要服务器和编程，在 GitHub 点几下，就能自己部署一套。


这个项目，和我之前分享过的 BettaFish，

功能都是获取各种热点新闻资讯，但是偏重不一样，

因为都挺有用的，所以在 GitHub 特别火，（两个加一起）已经连续霸榜了近一个月。


TrendRadar（获取新闻）
https://github.com/sansan0/TrendRadar

## 3.legado-with-MD3

这是知名项目开源阅读的一个修改版本，

它使用了谷歌的 Material Design 3 风格规范，同时对很多界面进行了优化。

比如信息流界面，原版的长这样，MD3 版本的长这样，看起来要舒服很多。


同时，原版的各种功能它也都有，

只需要导入规则，就可以搜索、查看各种小说、漫画。


如果你在使用原版的开源阅读，也可以试一下这个版本。



legado-with-MD3（阅读MD3版）
https://github.com/HapeLee/legado-with-MD3

## 4.winboat


这是一个开源的虚拟化工具，

可以让用户在 Linux 系统上，无缝、流畅的使用各种 Windows 软件。

和 Wine 通过转译来运行 Windows 软件不同，

它的原理是，在 Linux 系统上使用 Docker 运行一个完整的 Windows 系统，

把各种软件跑在容器内，然后使用 FreeRDP 来提供无缝的系统级窗口。


winboat 的使用非常简单，全程界面化操作，

并且几乎支持几乎所有 Windows 软件，

如果你是一个 Linux 用户，但是有使用 Windows 软件的需求，可以尝试一下。



winboat（Linux用Win软件）
https://github.com/TibixDev/winboat

## 5.goodtime

这是一个开源的手机番茄钟工具，

番茄工作法是一种非常简单的时间管理办法，

它的核心原理是专心工作 25 分钟，然后短暂休息五分钟恢复能量。

这种方式，可以让人在工作的时间内，集中注意力，以此来提高工作效率。


goodtime 就是使用了这个原理，

它的功能非常简单，打开之后点击屏幕，就开始 25 分钟倒计时，

倒计时完毕进入休息阶段。

你也可以自己手动控制，通过长按滑动选择的方式，

停止当前阶段、进入上一个或下一个阶段、增加 1 分钟本阶段的时间。

goodtime（番茄钟）
https://github.com/adrcotfas/goodtime

## 6.mimotion

在之前我曾经分享过，

微信的运动步数，可以通过 小米运动、Zepp Life 等运动软件进行同步。


而 Zepp Life 中的步数，非常容易就能修改，

这也代表着，可以通过修改 Zepp Life，从而达到自定义微信步数的目的。


mimotion 就是一个这样的项目，

并且，它部署起来也非常简单，只需要按照教程，在 GitHub 点几下即可，

部署完成后，你就可以任意调整自己的步数了。


mimotion（修改步数）
https://github.com/TonyJiangWJ/mimotion

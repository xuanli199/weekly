### 1.BettaFish

这是一个开源的舆情分析平台，可以帮助你破除信息检方，还原事件舆情原貌，并预测未来走向。

并且它使用起来也非常简单，只需要像聊天一样，提出来你要分析的内容，然后系统就会自动抓取分析国内外三十多家主流社媒和上百万大众评论的数据，并基于这些数据做分析，尽可能真实地生成报告。

比如某大学事件的舆情，使用它做分析，最后出来的报告是这样的，非常详细。

类似的舆情分析平台大部分都是企业级的专业收费软件。

BettaFish 作为一个开源项目，不但完全免费，做的也非常好。

[BettaFish：https://github.com/666ghj/BettaFish]((https://github.com/666ghj/BettaFish)

### 2. spleeter（声音分离）

这是一个开源的 desire 元分离库，可以使用它很轻松的在一段声音里分离出来其中的人声、音乐声、背景声。

不过 split 作为一个基础库，用起来比较复杂，但还好有很多人基于它做了二次开发。

比如这个叫 spliterGUI的工具，在原项目上加了 GUI，使用起来就要简单不少。

https://github.com/deezer/spleeter
https://spleetergui.com/

### 3. free-font（字体）

这是一个可商用的免费字体合集项目，其中收集了一千多款可商用的字体，覆盖了中文、英文、日语等常用的语言。

字体的类型也很丰富，黑体、楷体、宋体、艺术字、手写字各种样式都有。

不管是做什么工作，需要用到文字的地方都一定要注意，不要选没有授权和版权的字体。

https://github.com/jaywcjlove/free-font

### 4. toon（节省Token）

这是最近爆火的一个开源 AI 省 token 神器。

在使用 AI 大模型的时候，数据的传输效率往往会决定 AI 的性能和成本。

目前大多数人都是使用 JSON 的方式给 AI 发送数据。

JSON 对于人类来说非常易读易使用，但是对于 AI 来说，其中冗余的括号等语法却非常浪费 token。
而 token 设计了一种使用缩进来表示结构化数据的方式。使用 toon 格式能平均省下的大概30%到60%的 Token 使用量，不但能提升 AI 的性能和效率，还省了不少 Token 费用。

https://github.com/johannschopplich/toon

### 6. Everywhere（AI助手）

这是一个开源的交互式AI助手。

和其他的AI不一样的是，它可以随时看到你电脑屏幕上的内容。

你只需要正常的使用电脑，然后等到需要帮助的时候，按下快捷键把它召唤出来提出问题，它就能结合屏幕上的内容给你答案。

比如写代码的时候遇到报错，就可以召唤  everywhere出来问他这是什么错误，应该怎么解决。

https://github.com/DearVa/Everywhere

### 7. wechat-multi-open（多开）

这是一个开源的mac微信多开工具。

它的原理是复制微信应用并修改帮助 ID 让系统认为它是一个不同的应用，从而实现数据隔离和多开。

不过这种方式未来有可能会被微信禁止，大家按需使用。

https://github.com/nullbyte-lab/wechat-multi-open

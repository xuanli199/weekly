# 2025年12月27日-2026年1月3日


## 1.Green-Wall

2025 年过去，各大 APP 都出了年度报告，

但是 GitHub 并没有类似的功能，所以，我就找了两个开源项目，

一个叫 Green-Wall，一个叫 CyberGit，

两个项目的功能类似，都是读取你的 GitHub 数据，然后生成一份年度总结。

Green-Wall 的风格比较简约，CyberGit 更加极客风一点，

如果你是一个开源项目作者，可以给自己生成一份年报试一下。



GitHub 年报
https://github.com/Codennnn/Green-Wall
https://github.com/Amery2010/cybergit


## 2.docker-android

看名字就知道它的功能，

这是一个可以让你在 Docker 里，运行 Android 服务的项目，

并且能自定义安卓版本、镜像类型，并通过 scrcpy 进行界面操作。

如果你是一个安卓开发，需要不同版本的系统进行测试，

或者有其它需要使用多版本安卓的需求，那可能会对你有用。


docker-android
https://github.com/HQarroum/docker-android

## 3.Handy


这是一个开源的电脑语音输入工具，

它使用起来非常简单，按下快捷键，对着麦克风说话，

对应的内容就会被转换成文字，并添加到输入框里。


因为是一个开源项目，所以它语音转文字时，使用的也是开源模型，

首次打开的时候，必须要选择下载一个，

不同的模型，有不同的效果，我这里比较推荐 Whisper Turbo 模型，

并手动把语言设置为简体中文，

进入软件之后，如果对模型不满意，也可以在左下角切换。



电脑语音输入
https://github.com/cjpais/Handy

## 4.XMSLEEP


这是一个开源的安卓白噪音 APP，

白噪音是一种频率均匀的声音，

有让人在工作时集中注意力、睡觉时助眠等效果。


XMSLEEP 使用起来很简单，

打开之后，点击一个音效就会自动播放，

再点击其它音效，会多个声音叠加播放。

在左边侧边栏，能关闭指定的声音。

点击正在播放中的声音，可以调整音量大小。

点击右下角的钟表，能设置定时关闭。



白噪音
https://github.com/Tosencen/XMSLEEP

## 5.memos


这是一个开源的笔记项目，

它的功能很简单，就是让你随手记录自己想记的东西，

可以是长篇大论、可以是日记、也可以只是一句话。


所以，能把它当成日记软件、笔记软件、也可以当成团队的知识库。


和类似的商业软件相比，memos 完全开源、免费，支持自己部署到服务器，拥有极强的隐私保护。

如果你或者你的团队有类似的需求，可以了解一下。


开源笔记
https://github.com/usememos/memos

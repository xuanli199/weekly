# 1.everything-claude-code

当前市面上最强的通用 agent 的产品，我认为不是目前很火的 OpenClaw，而是   claude code。

但是会有一个问题，它的功能实在太多了，并且更新的很快，怎么配置才好用？应该装什么skills？

如何尽可能发挥它的能力上限，都是一个需要花很多时间研究的问题。

这种情况下直接抄冠军的作业就是一个很好的选择。

Everything-claude-code 这个项目是 Anthropic 黑客松冠军使用的 claude code 配置集合。这里面不是一些命令，也不是几个 skills，而是一整套工程化的配置。怎么优化 token，怎么内存持久化，有哪些好用的 MCP skill，一个开源项目全部解决。因为确实很好用，所以它在 GitHub 上目前足足有接近 90K 的 star。

everything-claude-code：https://github.com/affaan-m/everything-claude-code

# 2.NeriPlayer

这是一个开源的安卓音乐播放器，和普通播放器不同的是一个软件就可以听某云音乐、某b、youtube等多个来源的歌。

根据我的体验，它目前的各种功能已经很完善了，听歌、歌单、定时、歌词下载，各种常用功能都很齐全。

对于很多听歌人来说，最痛苦的就是自己喜欢的歌版权分布在不同平台，导致需要下载一大堆音乐软件。

NeriPlayer 目前支持的平台不是很多，但是理念很不错，未来如果能支持更多平台，那么将会解决一个很大的痛点。

NeriPlayer：https://github.com/cwuom/NeriPlayer

# 3.spank

这是一个很奇怪很抽象的科研项目。把它安装到 macbook 上之后，它会调用电脑上的加速传感器，然后只要拍打电脑，它就会发出奇怪的声音。

同时它还有好几种不同的模式，比如这个性感模式，千万不要在人多的时候使用。 Macbook 上因为传感器很多，所以这种抽象项目也很多。就比如这个叫 LidAngle Sensor的项目，使用之后只要开合屏幕，电脑就会嘎吱嘎吱响。

spank：https://github.com/taigrr/spank
LidAngleSensor：https://github.com/samhenrigold/LidAngleSensor

# 4.go-music-dl

这是一个开源的音乐下载器，它可以在一个网页上让我们听各大平台的歌，同时还能下载成本地MP3，功能很简单，但是很实用，很强大。

go-music-dl：https://github.com/guohuiyuan/go-music-dl
